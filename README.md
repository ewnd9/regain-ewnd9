# `regain-ewnd9`

Bootstraped from [`obenjiro/regain`](https://github.com/obenjiro/regain)

Overview / statistics / querying of all publicly available code from [`github.com/ewnd9`](https://github.com/ewnd9)

## Install

```sh
$ yarn install
```

## Usage

```sh
$ yarn sync-repos
$ yarn start:dev
```

## Deploy

```sh
$ yarn deploy
```
